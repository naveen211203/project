# Insurance Management System

## Project Overview
This project is an Insurance Management System developed using Java and MySQL. It allows managing insurance-related data such as policies, users, and transactions in a structured and efficient manner.

## Database Configuration
- **Database Name:** `insurancedb`
- Ensure that the database name is exactly the same as mentioned above to avoid configuration issues.

## Prerequisites
Before running this project, make sure the following requirements are met:

- MySQL is installed on your system.
- A MySQL server is running properly.
- Java Development Kit (JDK) is installed.
- Any required IDE (Eclipse, IntelliJ IDEA, or similar).

## Database Setup
- The required MySQL table creation scripts are provided in the file **`table-requirements.txt`**.
- If the tables are not already created in your database, execute the SQL scripts from this file in the `insurancedb` database.

## Steps to Run the Project
1. Install and start MySQL.
2. Create a database named `insurancedb`.
3. Execute the SQL queries available in `table-requirements.txt`.
4. Update database credentials (username and password) in the project configuration files if required.
5. Build and run the project using your preferred IDE or command line.

## Notes
- Ensure the database connection details are correct before running the application.
- Table structure must match the SQL scripts provided to avoid runtime errors.
